# Roadmap

This roadmap is intentionally short and contribution-oriented.

## Released

- `v0.2.x`
  - Streaming API (RSI, EMA, SMA)
  - Golden tests vs TA-Lib
  - Benchmark suite

- `v0.3.0` (released on 2026-02-26)
  - Gate #2: Streaming/Stateful API (real-time first)
  - Gate #3: Numeric validation and compatibility proof
  - Gate #7: API typing and developer UX
  - Meta gate #8: release checklist and publish

## Next focus (post-v0.3)

1. Expand indicator coverage and crypto-specific utilities.
2. Harden compatibility matrix and reproducibility automation.
3. Improve benchmark depth (larger datasets and scenario-based runs).
4. Add production-grade examples and strategy playbooks.

## How to contribute against roadmap

1. Pick one item.
2. Open or claim a guided issue.
3. Ship code + tests + short benchmark/docs update.
